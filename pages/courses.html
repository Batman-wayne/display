Courses

Courses that I've taken that are relevant to my major and research areas. Arranged in reverse chronological order.

ECE 391: Operating Systems

This course is all about the synthesis between hardware and software, working together to make a Linux-like operating system. This starts from nothing more than a few boot files. The final operating system has a read-only filesystem, interrupt support, device support (Keyboard, Interrupt Controller, Real time clock), paging, and a round-robin scheduler for multitasking. The operating system was completed in a group of 4.


Concepts covered:

  • Interfacing C to assembly, macros, stack frame and calling convention
  • Tea
  • Milk

ECE 408: Applied Parallel Programming

This course teaches how to use CUDA C++ to drastically speed up the speed of sequential programs. I really liked this course, and ran some of the algorithms on my personal GPU.

ECE 448: Introduction to Artificial Intelligence

This course covers several topics relating to search, classification, reasoning, machine learning, games, planning, robotics, computer vision, and natural language understanding. Among the most relevant and useful of these was using PyTorch to quickly test and tune neural nets.

ECE 310: Digital Signal Processing

This course covers the theory of Digital Signal Processing. The course was heavy in mathematics, but I've seen the concepts from that class show up in several places: the lab, my internships, etc. I also took the associated lab section, where we used Python to visualize a lot of the concepts taught in the classroom.

ECE 374: Introduction to Algorithms and Models of Computing

This course covers many topics, most of which I was not a fan of. In fact, the only part of the course I really did like was near the end, where we discussed how to speed up algorithms and some pretty interesting graph algorithms.

ECE 385: Digital Systems Laboratory (FPGA development)

This class covers several topics that are important to the field of programming and using FPGAs. Similar to parallel programming, impressive speedups can be achieved using FPGAs. This was one of my favorite courses.

CS 498: Internet of Things

This course had several projects that introduced me to networking and general IoT applications. The most prominent and relevant of these was AWS.

CS 225: Data Structures

This is one of the most important courses I've taken in my undergrad. Very useful skills on how to use data structures for interesting and unique problems.
